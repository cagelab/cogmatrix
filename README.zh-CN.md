# CogMatrix

面向神经科学实验数据管理的 GraphQL 服务。

## 项目概述

CogMatrix 旨在为神经科学实验数据（如受试对象、任务、会话、记录、标注与元数据）提供统一的组织、管理与查询能力。

项目受 alyx 启发，使用 Node.js + TypeScript 实现，并采用 GraphQL（而非仅 REST）来提供更灵活、面向研究者的数据访问方式。


## 为什么选择 GraphQL

- 按需获取字段，避免过取与欠取
- 更自然地表达复杂关系（实验 -> 会话 -> 记录 -> 标注）
- 单次查询聚合多源相关数据
- 借助 schema 自文档提升探索式查询体验
- 便于长期渐进演进且减少对既有分析流程的破坏

## 性能策略

该系统主要服务于科研场景下的交互式与批处理型工作流，不以极端 QPS 为目标。

优先级：

- 开发者与研究者查询体验优先于极限吞吐
- 日常数据探索场景下的低延迟交互查询
- 可预测、可理解的性能特征
- 优先保证数据建模的清晰性与可维护性

## 快速开始（当前骨架）

### 环境要求

- Node.js >= 20
- pnpm

### 安装依赖

```bash
pnpm install
```

### 开发运行

```bash
pnpm run start:dev
```

默认端口为 `3000`（可通过 `PORT` 环境变量覆盖）。

### 常用脚本

```bash
pnpm run build
pnpm run lint
pnpm run test
pnpm run start:prod
```
